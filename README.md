# Blog

Personal blog built with [Astro](https://astro.build/).

Live at: https://steve-z-wang.github.io/

## Tech Stack

- Astro 5
- MDX
- Tailwind CSS 4

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Adding Posts

Create a new folder in `src/content/posts/` with an `index.md` or `index.mdx` file.

## Deployment

Automatically deployed to GitHub Pages on push to `main`.
